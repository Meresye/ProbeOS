# ProbeOS

ProbeOS is a portable security toolkit firmware designed for the **M5StickC Plus 2**.
It transforms the device into a compact tool for wireless exploration, signal analysis, and security research.

The goal of ProbeOS is to provide a lightweight, fast, and modular environment for experimenting with wireless technologies directly from a pocket-sized device.

## Features

* WiFi scanning and signal exploration
* BLE device detection
* Wireless signal analysis tools
* Modular hacking utilities
* Lightweight firmware
* Simple on-device interface

## Hardware

ProbeOS is designed for:

* M5StickC Plus 2
* ESP32-based devices

## Project Goals

* Create a portable security toolkit
* Provide an intuitive interface
* Allow modular expansion of tools
* Keep the firmware fast and efficient
